1 基础知识

1.1 计算机网络的概念、组成、功能、分类

  1. 概念

    计算机网络一个将分散的、具有独立功能的计算机系统，通过通讯设备与线路连接起来，由功能完善的软件实现资源共享和信息传递的系统

  2. 功能

    数据通讯、资源共享、分布式处理、提高可靠性、负载均衡

  3. 组成

    • 组成部分

      硬件、软件、协议

    • 工作方式

      边缘部分：用户直接使用

      核心部分：为边缘部分服务

    • 功能组成

      通信子网：实现数据通讯————————————物理层、数据链路层、网络层

      资源子网：实现资源共享/数据通讯---------传输层、会话层、表示层、应用层

  4. 分类

    • 按分部范围分

      广域网、城域网、局域网、个人区域网

    • 按使用者分

      公用网、专用网

    • 按交换技术分

      电路交换、报文交换、分组交换

    • 按拓扑结构分

      总线型、星型、环形、网状型

    • 按传输技术分

      广播式网络、点对点网络

  1. 速率、带宽、吞吐量

    速率即数据率或称数据传输率或比特率

    带宽用来表示网络的通信线路传送数据的能力

    吞吐量表示在单位时间内通过某个网络的数据量

  1. 时延、时延带宽积、往返时间、利用率

2 物理层

2.1 物理层基本概念

物理层解决如何在连接各种计算机的传输媒体上的传输数据比特流

确定与传输媒体接口有关的特性:机械特性、电气特性、功能特性、规程特性

2.2 数据通信基础知识

  1. 三种通信方式

    单工、半双工、全双工

  2. 数据传输方式

    串行、并行

  3. 同步&异步

    同步：在传输数据时，需先发送1个或多个同步字符

    异步：发送方可以在任何时候发送数据

2.3 码元、速率、波特

码元是指用一个固定时长的信号波形，代表不同离散数值的基本波形。当码元的离散状态有M个时

此时码元为M进制码元

速率表示单位时间内传输的数据量。可以用码元传输速率和信息传输速率表示。

2.4 奈氏准则和香农定理

  1. 奈氏准则

    在理想低通条件下，为了避免码间干扰，极限码元传输速率为2W baud2W\ baud2W baud，www是信道带宽

  2. 香农定理

    信道的极限数据传输速率Wlog2(1+S/N)Wlog_2(1+S/N)Wlog2​(1+S/N)

2.5 编码&调制

编码：将信号转成数字信号

调制：将信号转成模拟信号

2.6 物理层传输介质

导向性传输介质

非导向性传输介质

2.6 物理层设备

中继器：对信号进行再生和还原

集线器：对信号进行再生放大转发

3 数据链路层

3.1 基本概念

负责通过一条链路从一个节点向另一个物理链路直接相连的相邻节点传送数据报

3.2 封装成帧&透明传输

  1. 封装成帧

    在一段数据的前后部分添加首部和尾部，这样就构成了一个帧。接收端在收到了物理层上交的比特流后，就能根据首部和尾部的标记，从收到的比特流中识别帧的开始和结束

  2. 透明传输

    不管所传数据是什么样的比特组合，都应当能够在链路上传送

  3. 封装成帧方法

    • 字符计数法

      帧首部使用一个计数字段来标明帧内字符数

    • 字符填充法

      字节填充法是采用一些特定的字符来定界一帧的开始和结束的方法，当标志字节出现在待传送的数据信息当中时，就会被误认为是帧的边界。解决这一问题的一种方法是，发送端的数据链路层在出现标志字节的数据段前插入另一个转义字符，而在接收端的数据链路层在把数据送往网络层之前删除这个插入的转义字符。这种技术称为字节填充(byte stuffing)

    • 零比特填充法

      在发送端，扫描整个信息字段，只要连续5个1，就立即填入1个0

      在接收端，发送连续5个1时，就把后面的0删除

    • 违规编码法

      用两个编码中不会出现的电平方式来标志起始和结束

3.3 差错控制

  1. 差错来源

    传输中的差错都是由于噪声引起的，主要分为两种

    • 全局性 由线路本身电气特性所产生的随机噪声

      ​ 解决方法：提高信噪比

    • 局部性 外界特定的短暂原因所造成的冲击噪声

      ​ 解决方法：利用编码技术

  2. 比特错差错控制

    • 检错编码

      奇偶校验、循环冗余码CRC

    • 纠错编码

      海明码

  3. CRC计算方法

    • 生成多项式阶数为R，信息码左移R位
    • 用生成多项式（二进制数）对信息码做除，得到R位的余数

  4. 海明码

    • 确定校验码位数rrr

      2r>=k+r+12^r>=k+r+1 2r>=k+r+1

      kkk为要发送的数据位数

    • 确定校验码和数据的位置

      校验码分别放在2i2^i2i的位置上(i=0,1,2...i=0,1,2...i=0,1,2...)
